## 1. HTML 기본 구조

### 1-1. 문서 기본 골격

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>페이지 제목</title>
    <link rel="icon" href="images/favicon.png" type="image/x-icon">
    <meta name="description" content="검색 엔진을 위한 페이지 설명">
</head>
<body>
    <!-- 본문 내용 -->
</body>
</html>
```

| 요소 | 설명 |
|------|------|
| `<!DOCTYPE html>` | HTML5 문서임을 선언. **문서 맨 위에 반드시 작성** |
| `<html lang="ko">` | HTML 문서의 루트 요소 |
| `<meta charset="UTF-8">` | 한글 등 다국어 문자 사용 |
| `<meta name="viewport">` | 반응형 웹을 위한 설정 |
| `<title>` | 브라우저 탭에 표시되는 제목 |
| `<link rel="icon">` | 파비콘(사이트 아이콘) 설정 |
| `<meta name="description">` | 검색 엔진용 페이지 설명 |

### 1-2. 띄어쓰기와 특수문자

- HTML에서 **스페이스바는 한 칸만** 적용된다.
- 여러 칸 띄우기: `&nbsp;` (Non-Breaking Space) 사용

```html
<p>띄어쓰기 5칸: &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</p>
```

**특수문자 입력 방법**

| 방법 | 예시 |
|------|------|
| 한글 자음 키 | `ㄱ`~`ㅎ` 키 → 특수문자 선택 (★ 등) |
| 엔티티 코드 | `&lt;` → `<`, `&gt;` → `>`, `&amp;` → `&` |
| 이모지 | `Win` + `.` 키 → 이모지 선택 |

### 1-3. CSS 기본 (내부 스타일)

```html
<style>
h1 {
    background-color: yellow;
    width: 50%;
}
</style>
```

## 2. 블록 요소 (Block Element)

### 2-1. 블록 요소의 특징

1. **한 줄 전체**를 차지한다.
2. 항상 **줄바꿈**이 발생한다.
3. **넓이(width), 높이(height)** 설정 가능
4. **margin(바깥 여백), padding(안 여백)** 설정 가능
5. 블록 요소 안에 인라인 요소 포함 가능 (블록 요소도 포함 가능)
6. **레이아웃 구성**에 주로 사용

### 2-2. 대표적인 블록 요소

| 태그 | 설명 |
|------|------|
| `<div>` | division, 구역(박스)을 나누는 태그 |
| `<p>` | 문단 태그 |
| `<h1>` ~ `<h6>` | 제목 태그 (숫자가 작을수록 큼) |
| `<ul>`, `<ol>`, `<li>` | 목록 태그 |
| `<pre>` | 공백·줄바꿈을 그대로 표현 |
| `<blockquote>` | 인용문 |
| `<address>` | 주소 정보 |
| `<dl>`, `<dt>`, `<dd>` | 용어 정의 목록 |

### 2-3. 제목과 문단

```html
<h1>가장 큰 제목</h1>
<h2>두 번째 제목</h2>
<p>문단 태그입니다. 줄바꿈이 됩니다.</p>
```

### 2-4. 인용문과 pre

```html
<blockquote>
    노력은 배신하지 않는다. 인용문을 나타내는 블록 요소
</blockquote>

<pre>
    pre 태그는 공백과 줄바꿈을
        그대로 표현합니다.
</pre>
```

### 2-5. 목록 (List)

**순서 없는 목록 (ul - unordered list)**

```html
<ul>
    <li>항목 1</li>
    <li>항목 2</li>
</ul>

<!-- 불릿 모양 변경 -->
<ul type="square">...</ul>
<ul type="none">...</ul>
```

**순서 있는 목록 (ol - ordered list)**

```html
<ol>
    <li>1번 항목</li>
    <li>2번 항목</li>
</ol>

<!-- 번호 형식 변경 -->
<ol type="A">...</ol>   <!-- A, B, C -->
<ol type="I">...</ol>   <!-- I, II, III -->
```

**용어 정의 목록 (dl - definition list)**

```html
<dl>
    <dt>말티즈</dt>
    <dd>사랑스럽고 시기심 많은 강아지</dd>
    <dt>요크셔테리어</dt>
    <dd>용맹하고 애교 많은 강아지</dd>
</dl>
```

### 2-6. 주소

```html
<address>
    서울시 구로구 182-13<br/>
    대륭포스트 2차 2층<br/>
    TEL: 02-838-1680
</address>
```

## 3. 인라인 요소 (Inline Element)

### 3-1. 인라인 요소의 특징

- **내용만큼만** 공간을 차지한다.
- **줄바꿈이 발생하지 않는다.**
- 옆으로 나란히 배치된다.
- `width`, `height` 직접 설정이 어렵다.

### 3-2. 글자 관련 인라인 태그

| 태그 | 설명 | 비고 |
|------|------|------|
| `<strong>` | 강조 (굵게) | **시맨틱 태그**, 웹접근성 O |
| `<em>` | 강조 (기울임) | **시맨틱 태그**, 웹접근성 O |
| `<b>` | 굵게 | 시각적 효과만 |
| `<i>` | 기울임 | 시각적 효과만 |
| `<mark>` | 형광펜 효과 | |
| `<small>` | 작은 글씨 | |
| `<del>` | 취소선 | |
| `<ins>` | 밑줄 | 접근성 O |
| `<u>` | 밑줄 | |
| `<sup>` | 윗첨자 | 예: X² |
| `<sub>` | 아랫첨자 | 예: CO₂ |

```html
<p>요즘은 <strong>반응형</strong> 웹이 대세이다.</p>
<p>요즘은 <em>반응형</em> 웹이 대세이다.</p>
<p>마크태그는 <mark>형광펜</mark> 역할</p>
<p>삭제선은 <del>취소선</del> 역할</p>
<h3>X<sup>2</sup> 윗첨자</h3>
<h3>CO<sub>2</sub> 아랫첨자</h3>
```

### 3-3. 링크 (a 태그)

```html
<!-- 같은 창에서 열기 -->
<a href="https://nate.com">네이트 가자</a>

<!-- 새 탭에서 열기 -->
<a href="https://nate.com" target="_blank">네이트 가자</a>

<!-- 같은 폴더의 파일 열기 -->
<a href="basic2.html" target="_blank">basic2 열기</a>

<!-- 상위 폴더의 파일 열기 -->
<a href="../0720/basic.html" target="_blank">basic 열기</a>
```

| 속성 | 설명 |
|------|------|
| `href` | 이동할 주소 (URL 또는 파일 경로) |
| `target="_blank"` | 새 탭(새 창)에서 열기 |

## 4. 인라인-블록 요소 & 미디어

### 4-1. 인라인-블록 (inline-block)

- **넓이·높이**를 가질 수 있지만 **줄바꿈은 안 됨**
- 대표 요소: `<img>`, `display: inline-block`으로 변경한 요소

```css
h3 {
    display: inline-block;
    vertical-align: middle;
}
```

### 4-2. 이미지 (img)

```html
<img src="images/dog1.jpg" width="300" alt="강아지사진1" title="강쥐1">
```

| 속성 | 설명 |
|------|------|
| `src` | 이미지 파일 경로 |
| `width` / `height` | 크기 |
| `alt` | 대체 텍스트 (접근성, **필수**) |
| `title` | 마우스 올렸을 때 툴팁 |

### 4-3. 비디오 (video)

```html
<!-- 자동 재생 (muted 필수) + 반복 -->
<video autoplay muted loop width="400">
    <source src="media/jungkook.mp4" type="video/mp4">
</video>

<!-- 컨트롤 버튼 표시 -->
<video controls width="400">
    <source src="media/jungkook.mp4" type="video/mp4">
</video>
```

| 속성 | 설명 |
|------|------|
| `autoplay` | 자동 재생 (`muted`와 함께 사용해야 함) |
| `muted` | 음소거 |
| `loop` | 반복 재생 |
| `controls` | 재생/일시정지 등 컨트롤 표시 |

### 4-4. 수평선

```html
<hr>
```

## 5. 표 (Table)

### 5-1. 기본 표 구조

```html
<table border="1">
    <tr>
        <td>1행 1열</td>
        <td>1행 2열</td>
    </tr>
    <tr>
        <td>2행 1열</td>
        <td>2행 2열</td>
    </tr>
</table>
```

| 태그 | 설명 |
|------|------|
| `<table>` | 표 전체 |
| `<tr>` | 행 (table row) |
| `<td>` | 일반 셀 (table data) |
| `<th>` | 제목 셀 (table header, 굵게·중앙정렬) |
| `<caption>` | 표 제목 |
| `<thead>` | 표 머리 영역 |
| `<tbody>` | 표 본문 영역 |
| `<colgroup>`, `<col>` | 열 스타일 지정 |

### 5-2. 표 CSS 스타일

```css
table {
    border: 1px solid #000;
    border-collapse: collapse;   /* 셀 테두리를 하나로 합침 */
    margin-left: auto;
    margin-right: auto;        /* 표 가운데 정렬 */
}

td, th {
    border: 1px solid #000;
    padding: 10px 20px;        /* 안 여백: 위아래 10px, 좌우 20px */
    text-align: center;
}
```

**padding 값 순서**

```css
padding: 10px;           /* 상하좌우 모두 10px */
padding: 10px 20px;        /* 상하 10px, 좌우 20px */
padding: 10px 20px 30px;   /* 위 10px, 좌우 20px, 아래 30px */
padding: 10px 20px 30px 40px; /* 위, 오른쪽, 아래, 왼쪽 */
```

### 5-3. 셀 병합

**가로 병합 (colspan)**

```html
<td colspan="2">2칸 합치기</td>
```

**세로 병합 (rowspan)**

```html
<td rowspan="2">2행 합치기</td>
```

### 5-4. 완성된 표 예제

```html
<div class="container">
    <h1>레드향</h1>
    <table>
        <caption>선물용과 가정용 상품 구성</caption>
        <colgroup>
            <col style="width:150px; background-color:#ccc;">
            <col style="width:200px;">
            <col style="width:150px;">
            <col style="width:150px;">
        </colgroup>
        <thead>
            <tr style="background-color:#ccc;">
                <th>용도</th>
                <th>중량</th>
                <th>개수</th>
                <th>가격</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td rowspan="2">선물용</td>
                <td>3kg</td>
                <td>11~16과</td>
                <td>35,000</td>
            </tr>
            <tr>
                <td>5kg</td>
                <td>18~26과</td>
                <td>52,000</td>
            </tr>
            <tr>
                <td rowspan="2">가정용</td>
                <td>3kg</td>
                <td>11~16과</td>
                <td>30,000</td>
            </tr>
            <tr>
                <td>5kg</td>
                <td>18~26과</td>
                <td>47,000</td>
            </tr>
        </tbody>
    </table>
</div>
```

### 5-5. 컨테이너 가운데 정렬

```css
.container {
    width: 800px;
    border: 1px solid #000;
    margin-left: auto;
    margin-right: auto;   /* 좌우 auto → 화면 중앙 정렬 */
}
```

## 6. 블록 vs 인라인 비교

| 구분 | 블록 요소 | 인라인 요소 |
|------|-----------|-------------|
| 공간 | 한 줄 전체 차지 | 내용만큼만 차지 |
| 줄바꿈 | 발생함 | 발생 안 함 |
| width/height | 설정 가능 | 설정 어려움 |
| margin/padding | 설정 가능 | 제한적 |
| 대표 태그 | div, p, h1~h6, ul, ol | a, strong, em, img, span |

## 8. 핵심 키워드 정리

```
<!DOCTYPE html>     → HTML5 문서 선언
&nbsp;              → 공백 (Non-Breaking Space)
&lt; &gt;           → < > (엔티티 코드)
target="_blank"     → 새 탭 열기
border-collapse     → 표 테두리 합치기
colspan             → 가로 셀 병합
rowspan             → 세로 셀 병합
autoplay + muted    → 비디오 자동 재생
alt                 → 이미지 대체 텍스트 (접근성)
```
