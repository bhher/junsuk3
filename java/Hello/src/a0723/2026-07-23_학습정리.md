## 1. 조건문 (if)

조건이 참(`true`)일 때만 특정 코드를 실행한다.

### if (단순 조건)

```java
int age = 20;

if (age >= 19) {
    System.out.println("성인입니다.");
}

System.out.println("프로그램 종료");
```

> `if` 블록 밖 코드는 조건과 관계없이 항상 실행된다.

### if ~ else (둘 중 하나)

```java
int score = 55;

if (score >= 60) {
    System.out.println("합격");
} else {
    System.out.println("불합격");
}
```

### if ~ else if ~ else (여러 조건)

위에서부터 조건을 순서대로 검사하고, 처음 참인 블록만 실행한다.

```java
int score = 88;

if (score >= 90) {
    System.out.println("A학점");
} else if (score >= 80) {
    System.out.println("B학점");
} else if (score >= 70) {
    System.out.println("C학점");
} else if (score >= 60) {
    System.out.println("D학점");
} else {
    System.out.println("F학점");
}
```

| 점수 범위 | 학점 |
|-----------|------|
| 90 이상 | A |
| 80 ~ 89 | B |
| 70 ~ 79 | C |
| 60 ~ 69 | D |
| 60 미만 | F |

### 중첩 if

`if` 안에 또 다른 `if`를 넣을 수 있다. (여러 조건을 동시에 만족해야 할 때)

```java
int age = 25;
boolean member = true;

if (age >= 19) {
    if (member) {
        System.out.println("성인회원입니다.");
    }
}
```

### 논리 연산자 (`&&`, `||`)

| 연산자 | 의미 | 예시 |
|--------|------|------|
| `&&` | AND (둘 다 참) | `age >= 20 && age < 30` → 20대 |
| `\|\|` | OR (하나라도 참) | `age < 19 \|\| age >= 65` → 할인 대상 |

```java
int age = 17;

if (age < 19 || age >= 65) {
    System.out.println("할인대상");
} else {
    System.out.println("성인요금입니다.");
}
```

## 2. Scanner (키보드 입력)

`java.util.Scanner`로 사용자 입력을 받는다.

```java
import java.util.Scanner;

Scanner sc = new Scanner(System.in);

System.out.print("점수입력 : ");
int score = sc.nextInt();  // 키보드로 숫자 입력 → score에 저장

sc.close();  // 사용 후 닫기
```

### 합격 / 불합격 판정

```java
System.out.print("점수입력 : ");
int score = sc.nextInt();

if (score >= 60) {
    System.out.println("합격");
} else {
    System.out.println("불합격");
}
```

### 양수 판별

```java
System.out.print("숫자 입력 : ");
int num = sc.nextInt();

if (num > 0) {
    System.out.println(num + " 양수입니다.");
} else {
    System.out.println(num + " 양수아닙니다.");
}
```

### 여러 점수 입력 → 총점 / 평균

```java
System.out.print("국어점수 : ");
int kor = sc.nextInt();

System.out.print("영어점수 : ");
int eng = sc.nextInt();

System.out.print("수학점수 : ");
int math = sc.nextInt();

int sum = kor + eng + math;
double avg = (double) sum / 3;  // 형변환 → 실수 나눗셈

System.out.println("총점 : " + sum);
System.out.printf("평균 : %.1f%n", avg);  // 소수점 첫째 자리까지 출력
```

> **형변환**: `(double) sum / 3` → 정수 나눗셈이 아니라 실수 나눗셈이 된다.

## 3. 배열 (Array)

같은 타입의 여러 값을 **하나의 이름**으로 묶어 관리한다.

### 배열을 사용하지 않는 경우

변수가 많아지면 코드가 길고 반복이 많다.

```java
int score1 = 90;
int score2 = 85;
int score3 = 78;
int score4 = 95;
int score5 = 88;

System.out.println(score1);
System.out.println(score2);
// ...
```

### 배열 선언 및 초기화

```java
int[] score = {90, 85, 78, 95, 88};

System.out.println(score[0]);  // 90 (인덱스 0부터 시작)
System.out.println(score[1]);  // 85
System.out.println(score[4]);  // 88
```

| 개념 | 설명 |
|------|------|
| 인덱스 | 배열 요소의 위치 (0부터 시작) |
| `score.length` | 배열 크기 (요소 개수) |
| `score[i]` | i번째 요소 접근 |

### for문으로 배열 순회

```java
int[] score = {90, 85, 78, 95, 88};

for (int i = 0; i < score.length; i++) {
    System.out.println(i + "번째 인덱스" + score[i]);
}
```

> `score.length`를 쓰면 배열 크기가 바뀌어도 반복 조건을 수정할 필요가 없다.

## 4. Scanner + 배열 활용

### 배열에 점수 저장 후 총점 / 평균

```java
int[] score = new int[3];  // 크기 3인 배열 생성

System.out.print("국어점수 : ");
score[0] = sc.nextInt();

System.out.print("영어점수 : ");
score[1] = sc.nextInt();

System.out.print("수학점수 : ");
score[2] = sc.nextInt();

int sum = score[0] + score[1] + score[2];
double avg = (double) sum / score.length;
```

### for문으로 입력 + 합계 계산

```java
int[] score = new int[3];
int sum = 0;

for (int i = 0; i < score.length; i++) {
    System.out.print((i + 1) + "번째 점수 : ");
    score[i] = sc.nextInt();
    sum += score[i];
}

double avg = (double) sum / score.length;

System.out.println("총점 : " + sum);
System.out.printf("평균 : %.1f%n", avg);
```

> 입력을 for문으로 처리하면 과목 수가 늘어나도 코드 수정이 거의 없다.
