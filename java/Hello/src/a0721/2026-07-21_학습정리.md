# Java 학습 정리 (2026-07-21)

## 1. 변수 (Variable)

**변수**는 데이터를 저장하는 공간이다.

```java
String name1 = "허준석";  // 문자열
int age = 30;              // 정수
double height = 175.5;     // 실수

System.out.println("이름 : " + name1);
System.out.println("나이 : " + age);

int score = 80;
score = 100;  // 변수 값 변경 가능
```

### 변수 명명 규칙

| 규칙 | 예시 | 설명 |
|------|------|------|
| 숫자로 시작 불가 | `int 1age;` ❌ | 변수명은 문자로 시작해야 함 |
| 예약어 사용 불가 | `int class;` ❌ | Java 키워드는 변수명으로 사용 불가 |
| 공백 사용 불가 | `int my age;` ❌ | 공백 없이 작성 |
| `_`, `$`로 시작 가능 | `int _var2;`, `int $var3;` ✅ | 특수문자는 `_`, `$`만 허용 |
| `#` 등 기타 특수문자 | `int #var4;` ❌ | `_`, `$` 외 특수문자 불가 |

## 2. 자료형 (Data Types)

### 정수형

| 타입 | 크기 | 범위 | 예시 |
|------|------|------|------|
| `byte` | 1 byte | -128 ~ 127 | `byte byteVar = 127;` |
| `short` | 2 byte | -32,768 ~ 32,767 | `short shortVar = 32767;` |
| `int` | 4 byte | -2,147,483,648 ~ 2,147,483,647 | `int age = 25;` |
| `long` | 8 byte | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 | `long var6 = 10000000000L;` |

> `int` 범위를 초과하는 정수는 `long` 타입을 사용하고, 끝에 `L` 또는 `l` 접미사를 붙인다.

### 실수형

| 타입 | 크기 | 예시 |
|------|------|------|
| `float` | 4 byte | `float floatVar = 3.14F;` |
| `double` | 8 byte | `double height = 175.5;` |

> `float` 타입은 끝에 `F` 또는 `f` 접미사를 붙인다.

### 문자형

```java
char charVar = 'A';          // 영문자
char koreanChar = '가';      // 한글
char unicodeChar = '\u0061'; // 유니코드 (a)
```

> 유니코드는 인터넷에서 검색하여 사용할 수 있다.

### 논리형

```java
boolean isTrue = true;
boolean isFalse = false;
boolean result = (5 > 3);  // true
```

### 참조 자료형

```java
String name = "홍길동";  // 문자열
```

### 상수 (Constant)

`final` 키워드를 사용하면 값 변경이 불가능한 상수를 선언할 수 있다.

```java
final double PI = 3.14159;
final int MAX_SCORE = 100;

// PI = 1.19;  // 컴파일 에러 - 상수는 변경 불가
```

## 3. 산술 연산자 (Arithmetic Operators)

```java
int a = 10;
int b = 3;

a + b   // 13  (덧셈)
a - b   // 7   (뺄셈)
a * b   // 30  (곱셈)
a / b   // 3   (나눗셈)
a % b   // 1   (나머지)
```

### 정수 나눗셈 vs 실수 나눗셈

```java
int x = 10;
int y = 3;

int result1 = x / y;              // 3       (정수 나눗셈)
double result2 = x / y;           // 3.0     (정수 나눗셈 후 double 변환)
double result3 = (double) x / y;  // 3.333.. (형변환 → 실수 나눗셈)
```

> **형변환(업캐스팅)**: 정수를 실수로 변환하면 실수 나눗셈이 수행된다.

### 나머지 연산자 (`%`) 활용

```java
int num = 15;

num % 2   // 2의 배수 확인 (0이면 배수)
num % 3   // 3의 배수 확인
num % 5   // 5의 배수 확인

// 짝수/홀수 판별
if (num % 2 == 0) {
    System.out.println(num + " 짝수입니다.");
} else {
    System.out.println(num + " 홀수입니다.");
}
```

## 4. 증감 연산자 (Increment / Decrement)

### 전위 증가 (`++변수`)

변수를 **먼저 증가**시킨 후 값을 사용한다.

```java
int a = 5;
a++;           // a = 6

int b = ++a;   // a를 먼저 7로 증가 → b = 7
// a = 7, b = 7
```

### 후위 증가 (`변수++`)

**현재 값을 먼저 사용**한 후 변수를 증가시킨다.

```java
int c = 5;
int d = c++;   // d = 5 (현재값 할당) → c = 6 (이후 증가)
// c = 6, d = 5
```

### 전위 감소 (`--변수`)

변수를 **먼저 감소**시킨 후 값을 사용한다.

```java
int e = 10;
int f = --e;   // e를 먼저 9로 감소 → f = 9
// e = 9, f = 9
```

### 후위 감소 (`변수--`)

**현재 값을 먼저 사용**한 후 변수를 감소시킨다.

```java
int g = 10;
int h = g--;   // h = 10 (현재값 할당) → g = 9 (이후 감소)
// g = 9, h = 10
```

### 전위 vs 후위 비교

| 연산자 | 동작 순서 | 예시 (`a = 5`) |
|--------|-----------|----------------|
| `++a` (전위) | 증가 → 사용 | `b = ++a;` → a=6, b=6 |
| `a++` (후위) | 사용 → 증가 | `b = a++;` → a=6, b=5 |
| `--a` (전위) | 감소 → 사용 | `b = --a;` → a=4, b=4 |
| `a--` (후위) | 사용 → 감소 | `b = a--;` → a=4, b=5 |
