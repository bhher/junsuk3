## 1. 박스 모델 (box.html)

### 1-1. 블록 요소의 크기

- 블록 요소는 **width(넓이)**, **height(높이)** 를 지정할 수 있다.
- 단위: `px`(고정), `%`(부모 기준 비율)

```css
.box1 {
    width: 200px;
    height: 200px;
    background-color: blueviolet;
}
.box2 {
    width: 25%;   /* 부모 넓이의 25% */
    height: 200px;
}
```

### 1-2. padding / margin

| 속성 | 의미 |
|------|------|
| `padding` | **안여백** (테두리 안쪽) |
| `margin` | **바깥여백** (테두리 바깥) |

```css
.box3 {
    width: 200px;
    padding-left: 20px;  /* 왼쪽 안여백 → 전체 넓이 220px */
    margin-left: 50px;   /* 왼쪽 바깥여백 */
    margin-top: 50px;
}
```

### 1-3. 짧은 표기법 (단축 속성)

```css
padding: 10px;                 /* 상하좌우 모두 10px */
padding: 10px 20px 30px 40px;  /* 위 → 오른쪽 → 아래 → 왼쪽 (시계방향) */
margin: 10px 20px 30px;        /* 위 10 / 좌우 20 / 아래 30 */
```

### 1-4. 박스 가운데 정렬

```css
.box4 {
    width: 200px;
    margin: 0 auto;  /* 위아래 0, 좌우 auto → 가로 중앙 */
}
```

## 2. 색상 (color.html)

### 2-1. 색상 표현 방법

| 방법 | 예시 | 설명 |
|------|------|------|
| 색상 이름 | `color: red;` | 영문 색상명 |
| 16진수 | `color: #ff0000;` 또는 `#f00;` | `#RGB` / `#RRGGBB` |
| rgb | `color: rgb(255, 0, 0);` | 빨강·초록·파랑 (0~255) |
| rgba | `background-color: rgba(0,0,0,0.5);` | 마지막 값 = **투명도(alpha)** |

### 2-2. alpha(투명도)

- `rgba`의 네 번째 값: **0 ~ 1**
- `0` = 완전 투명, `1` = 완전 불투명, `0.5` = 50% 반투명

```css
.c5 {
    background-color: rgba(0, 0, 0, 0.5);  /* 반투명 검정 */
}
.c6:hover {
    background-color: rgba(255, 0, 0, 0.5); /* 마우스 올리면 반투명 빨강 */
}
```

## 3. 배경 (background1.html / background2.html)

### 3-1. 시맨틱 태그

| 태그 | 역할 |
|------|------|
| `<header>` | 머리글(헤더) |
| `<section>` / `<main>` | 본문(메인) |
| `<footer>` | 바닥글(푸터) |

의미 있는 태그를 **시맨틱 태그**라고 한다.

### 3-2. 배경 관련 속성

| 속성 | 설명 | 주요 값 |
|------|------|---------|
| `background-color` | 배경색 | 색상값 |
| `background-image` | 배경 이미지 | `url(경로)` |
| `background-repeat` | 반복 여부 | `no-repeat`, `repeat` 등 |
| `background-position` | 위치 | `right top`, `50% 50%`, `70% 50%` |
| `background-size` | 크기 | `cover`(채움), `contain`(맞춤) |
| `background-attachment` | 고정 여부 | `fixed`(스크롤해도 고정), `scroll` |

```css
header {
    background-color: aqua;
    background-image: url(images/tree.png);
    background-repeat: no-repeat;
    background-position: 70% 50%;
}
section {
    background-image: url(images/top.jpg);
    background-size: cover;  /* 영역을 꽉 채움 */
}
```

### 3-3. 배경 이미지 여러 개

쉼표(`,`)로 구분한다.

```css
footer {
    background-image: url(images/muzi.png), url(images/dot.png);
    background-repeat: no-repeat, no-repeat;
    background-position: left top, right bottom;
}
```

### 3-4. 단축 속성 (background)

```css
/* color image repeat position / size */
background: aqua url(images/back1.jpg) no-repeat 50% 50% / cover;
```

### 3-5. 전체 화면 섹션

```css
section {
    width: 100%;
    height: 100vh;  /* 뷰포트 높이 100% */
}
```

## 4. 폰트 (font1.html / font2.html / font3.html)

### 4-1. Google Fonts 사용법

1. Google Fonts에서 폰트 검색 (예: Noto Sans KR, Nanum Gothic)
2. Get font → Get embed code
3. `@import` 코드를 `<style>` 안에 복사

```css
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+KR:wght@100..900&display=swap');

.f1 {
    font-family: "Noto Sans KR", sans-serif;
    font-size: 32px;
    font-weight: 100;  /* 얇게 */
}
.f4 {
    font-weight: 900;  /* 아주 굵게 */
}
```

### 4-2. font-weight (글자 굵기)

| 값 | 의미 |
|----|------|
| `100` | Thin (가장 얇음) |
| `400` / `normal` | Regular (보통) |
| `700` / `bold` | Bold (굵게) |
| `800` ~ `900` | Extra Bold / Black |

### 4-3. 기타 글꼴 속성

| 속성 | 설명 | 예시 |
|------|------|------|
| `font-style` | 기울임 | `italic`, `normal` |
| `font-variant` | 소문자→작은 대문자 | `small-caps` |
| `line-height` | 줄간격 | `1`, `2`, `2.5` |
| `letter-spacing` | 자간(글자 간격) | `5px`, `normal` |

### 4-4. font 단축 속성

```css
.font3 {
    font: bold 20px/2.5 "궁서";
    /* 굵기  크기/줄간격  글씨체 */
}
```

## 6. 핵심 키워드 정리

```
padding / margin     → 안여백 / 바깥여백
margin: 0 auto;      → 가로 가운데 정렬
#f00 / rgb / rgba    → 색상 표현 (rgba는 투명도)
background-size: cover → 배경을 영역에 꽉 채움
background-attachment: fixed → 스크롤해도 배경 고정
100vh                → 화면 높이 전체
font-weight          → 글자 굵기 (100~900)
line-height          → 줄간격
letter-spacing       → 자간
font: bold 20px/2.5 "궁서" → font 단축 속성
```
