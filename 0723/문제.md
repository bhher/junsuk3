# CSS 연습 문제 (0723)

오늘 배운 내용(박스 모델 · 색상 · 배경 · 폰트)을 바탕으로 한 연습 문제입니다.

---

## 1단계: 객관식 (5문제)

**1.** `padding`과 `margin`의 차이로 맞는 것은?

- ① padding은 바깥여백, margin은 안여백
- ② padding은 안여백, margin은 바깥여백
- ③ 둘 다 안여백이다
- ④ 둘 다 바깥여백이다

**2.** 박스를 화면 가로 가운데에 두려면?

- ① `margin: auto 0;`
- ② `margin: 0 auto;`
- ③ `padding: 0 auto;`
- ④ `width: auto;`

**3.** `rgba(0, 0, 0, 0.5)`에서 `0.5`의 의미는?

- ① 색상 밝기 50%
- ② 가로 크기 50%
- ③ 투명도(alpha) 50% (반투명)
- ④ 글자 크기 50%

**4.** `background-size: cover;`의 역할은?

- ① 이미지를 원래 크기로 표시
- ② 이미지를 반복해서 채움
- ③ 이미지가 영역을 꽉 채우도록 확대/축소
- ④ 이미지를 숨김

**5.** `font: bold 20px/2.5 "궁서";`에서 `2.5`는 무엇을 의미할까?

- ① font-weight
- ② font-size
- ③ line-height (줄간격)
- ④ letter-spacing
